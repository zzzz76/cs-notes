### 1. 分数的表示和化简

分数的四则运算是指，给定两个分数的分子和分母，求他们加减乘除的结果。

我们使用假分数来表示分数，无论分子比分母大或者小，都保留其原数，使用如下结构存储分数：

```java
class Fraction {
  int up, down;
}
```

其中假分数的表示需要满足以下规定：

* 用分子为负来表示分数为负，分母始终非负。
* 用分子为0、分母为1来表示分数为0
* 分子和分母没有除1以外的公约数。



对于输入的分数，我们需要将其进行格式化，使其能够满足我们的假分数表示，从而能够支持规范的四则运算。

```java
public Fraction(int up, int down) {
  if (down < 0) {
    up *= -1;
    down *= -1;
  }
  
  if (up == 0) {
    down = 1;
  } else {
    int d = gcd(abs(up), abs(down));
    up /= d;
    down /= d;
  }
  this.up = up;
  this.down = down;
}
```

### 平衡二叉树的概念

平衡二叉树仍然是一颗二叉查找树，只是在其基础上增加了“平衡”的限制。所谓平衡是指，对于平衡二叉树的任意结点，其左子树与右子树的高度差不超过1，其中左子树与右子树的高度之差称为该结点的平衡因子。

为何要"平衡"？

需要保证树的高度在每次操作之后，仍然能够保持$O(logn)$，这样能让查询复杂度保持$O(logn)$。

链式存储结构：

```java
class Node {
    int data; //权值
    int height; //高度
    Node left;
    Node right;
}
```

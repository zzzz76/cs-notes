# 3.4 日期处理

## codeup 1928

**题目描述：**

求两个日期之间的天数，如果两个日期是连续的，那么规定它们之间的天数为1天

**输入格式：**

有多组数据，每组数据有两行，分别表示两个日期，格式为YYYYMMDD

**输出格式：**

每组数据输出，日期差值

**思略：**

1. 按照顺序交换日期
2. 令开始日期不断加1，直到结束日期
3. 用二维数组存放平年日历(365)和闰年日历(366)

**代码：**

```java
import java.util.*;

public class C1928 {

    public static int[][] months = {
            {0, 0}, {31, 31}, {28, 29}, {31, 31}, {30, 30}, {31, 31}, {30, 30},
            {31, 31}, {31, 31}, {30, 30}, {31, 31}, {30, 30}, {31, 31}
    };

    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);

        while(in.hasNext()) {
            int p1 = in.nextInt();
            int p2 = in.nextInt();
            if (p1 > p2) {
                int temp = p1;
                p1 = p2;
                p2 = temp;
            }

            //进行日期分解
            int year1 = p1 / 10000;
            int month1 = p1 % 10000 / 100;
            int date1 = p1 % 10000 % 100;

            int year2 = p2 / 10000;
            int month2 = p2 % 10000 / 100;
            int date2 = p2 % 10000 % 100;

            int count = 0;
            while (p1 < p2) {
                //开始走表
                count++;
                date1++;
                if (date1 > months[month1][isLeap(year1)]) {
                    date1 = 1;
                    month1++;
                    if (month1 > 12) {
                        month1 = 1;
                        year1++;
                    }
                }
                p1 = year1 * 10000 + month1 * 100 + date1;
            }

            System.out.println(count);
        }

    }

    // 判断是否是闰年
    public static int isLeap(int year) {
        // 4年一闰
        if ((year % 4 == 0 && year % 100 != 0) || year % 400 == 0) {
            return 1;
        } else {
            return 0;
        }
    }

}

```

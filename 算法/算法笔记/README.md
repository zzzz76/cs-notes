## 算法笔记

### 前言

本文用 Java 作为描述语言，总结了常见的算法题型和解题技巧，参考资料：

* 《算法笔记.胡凡》
* 《Java核心技术 卷2 高级特性 原书第10版》
* 《王道论坛-2020年数据结构考研复习指导》



简单说说如何刷题：

刷题是一个刻意训练的过程，其中训练分为三项，需求、建模、精化。首先，对给出的问题，进行快速的反应和准确的分析，将问题需求转化为技术需求；接着在理解需求的基础上，以构建系统为目标，先用规范的语言对系统进行描述，形成抽象的系统；当然，建模并非一次性完成的，需要不断的进行模型迭代，而这个过程就是精化的过程。最终，我们可以使用具体的编程语言实现系统，并利用给出的测试数据，完成对模型的检验。
